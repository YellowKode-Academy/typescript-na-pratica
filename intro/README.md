# Introdução — Setup do Projeto

Pasta de referência do capítulo introdutório do livro **TypeScript na Prática**.

## Pré-requisitos

- Node.js 22 ou superior
- npm 10 ou superior
- VS Code com a extensão TypeScript (já vem instalada por padrão)
- Docker (para rodar o PostgreSQL localmente no capítulo 9)

## Setup inicial

```bash
npm install
npm run dev
```

A partir do capítulo 2, o projeto começa a migração incremental para TypeScript. Cada pasta `cap-XX` contém o estado do código ao final daquele capítulo.
